  React Router 是一个基于 React 之上的路由库，可以实现在应用中快速地添加视图和数据流，同时保持页面与 URL 间的同步。

  在统开发中的路由，是由服务端根据不同的用户请求地址 URL，返回不同内容的页面，而前端路由则将这些任务通过 JS 在浏览器端完成。react-router 是官方指定和维护的 React 路由库，它通过管理 URL，实现组件间切换，和状态 (state) 的变化。
