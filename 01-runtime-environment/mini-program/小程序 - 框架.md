### mina 官方框架

```
[mina 官方框架]

	: 框架提供了自己的视图层描述语言 WXML 和 WXSS, 以及基于 JavaScript 的逻辑层框架
  : 在视图层与逻辑层间提供了数据传输和事件系统，可以让开发者可以方便的聚焦于数据与逻辑上。
```
